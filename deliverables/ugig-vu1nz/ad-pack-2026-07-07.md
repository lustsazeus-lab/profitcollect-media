# vu1nz Starter Ad Creative Pack

Prepared for the UGig task: "Create High-Quality Ads for vu1nz.com - Meta, Reddit, Google Ads - $1 Test Gig"

## Campaign Summary

vu1nz helps GitHub-heavy teams catch CI/CD workflow and package supply-chain risks before they merge. The campaign should position vu1nz as an additional security layer that complements Dependabot, CodeQL, Snyk, and Semgrep by scanning the parts those tools often do not cover: GitHub Actions workflow configuration and newly added package diffs.

Primary CTA: Scan your public GitHub repo free.

Secondary CTA: Start a 14-day free trial.

## Target Audience

- Software engineers who review PRs and maintain GitHub Actions workflows.
- DevOps and platform engineers responsible for CI/CD security.
- Security engineers and AppSec teams watching supply-chain risk.
- CTOs and technical founders at GitHub-native teams.
- Open-source maintainers who want lightweight workflow and dependency-change checks.

## Messaging Pillars

1. Breach prevention: Your app code may not be the weak point. Your workflow and dependency changes can be.
2. Developer utility: Paste a public GitHub repo and get CI/CD findings fast.
3. Competitive gap: Dependabot checks known package CVEs. CodeQL checks app code. vu1nz scans workflow and package-diff risks.
4. Fast install: Install one GitHub App and get PR check runs without maintaining YAML.
5. Open-source trust: MIT-licensed, auditable, forkable, and built for teams that ship.
